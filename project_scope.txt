1. A Fitness App
	1.1 Views
		1.1.1 Workout View
			1.1.1.1 Monthly view: show all days in a month, and workout snippet for each day
			1.1.1.2 Weekly view: show all days in a week, and workout snippet for each day
			1.1.1.3 Daily view: show all workouts in a day, sorted by time
				1.1.1.3.1 Instructional View: show the current days workouts in tutorial videos 
			1.1.1.4 Future view: show all upcoming workout days and workouts as a list 
		1.1.2 Meal View
			1.1.2.1 Monthly view: show all days in a month, and suggested meals snippet for each day
			1.1.2.2 Weekly view: show all days in a week, and suggested meals snippet for each day
			1.1.2.3 Daily view: show all suggested meals in a day, sorted by time
				1.1.2.3.1 Nutritional Facts View: show daily nutritional facts
			1.1.3.4 Future view: show all upcoming meals as a list
				1.1.3.4.1 Predicted Weight View: Show predicted weight gain/loss
			
	1.2 Social (optional)
		1.3.1 Add, view, or hide other profiles
		1.3.2 Display badges and achievements
		1.3.3 If verified can post articles and links on profile
